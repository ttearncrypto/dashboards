App Guide

Categories

Select a category from the top tabs to browse specific types of projects.

Search & Filter

Use the search bar to find projects by name. Click on any tag below a project to see others with the same tag.

Sort

Tap the icon in the header to sort the current list by date or name.

Project Actions

Tap "View Project" or "Read More" to open the project link. For some categories, you can tap the icon to copy the link.

Bottom Navigation

Use the bottom bar for quick access to Learn (), Chat (), News (), and Settings ().
